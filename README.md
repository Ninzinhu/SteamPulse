# SteamPulse

Radar de preço interno para Steam com notificações push no desktop.

## Estrutura inicial

- `src/` código fonte TypeScript
- `dist/` saída compilada

## Scripts

- `npm run dev` - desenvolve com ts-node-dev
- `npm run build` - compila TS
- `npm start` - executa app compilado
- `npm test` - executa testes

## Tecnologias

Node.js, TypeScript, Fastify/Express, BullMQ, Redis, PostgreSQL, Puppeteer, Web Push
